הנפשה

מושגים:

  • Timeline - ציר הזמן.
  • Animation Clip - סדרת שינויי מאפיינים על ציר-הזמן.
  • Keyframe - רגע על ציר-הזמן שבו קובעים את ערכי המאפיינים.
  • Animator Controller - מכונת מצבים של מעברים בין קטעי הנפשה.
  • Animator Component - רכיב המקשר בקר-הנפשה לעצם-משחק.

סוגי הנפשה:

  • Bone Animation - הנפשה ע"י שינוי מאפיינים של תת-עצמים המייצגים חלקי-גוף.
  • Sprite Animation - הנפשה ע"י שינוי התמונה של ה-Sprite Renderer.
    Sprite sheet - אותה דמות במצבים שונים היוצרים אשליה של תנועה.

  • אירועים - Events -

ממשק משתמש

האתגר: להשאיר את הפקדים באותו מקום על המסך, בלי תלות במיקום המצלמה או בגודל המסך.

מושגים:

  • UI = User Interface = ממשק משתמש.
  • Canvas - לוח חלק להצגת פקדי UI.
  • RectTransform - יורש את Transform ומוסיף עוגנים.
  • Anchor - עוגן - מקשר פקד UI לנקודה מסוימת במסך.

  • SceneManager - טוען הסצינות
  • Singleton - דגם עיצוב לשמירת מידע בין סצינות
