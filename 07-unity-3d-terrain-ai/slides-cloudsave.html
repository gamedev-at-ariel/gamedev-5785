שמירת נתוני שחקן - Unity CloudSave

אראל סגל-הלוי

המטרה

אנחנו רוצים לשמור את הנתונים של שחקן מסויים לאורך זמן, למשל כדי לעקוב אחר ההתקדמות שלו.

דרכים אפשריות

  1. בזיכרון - ייעלם לאחר שהשחקן סוגר את המשחק;
  2. בקובץ במחשב המקומי - ייעלם אם השחקן עובר לשחק במחשב אחר ;
  3. במסד נתונים - דורש לנהל שרת פרטי.

הפתרון של יוניטי - CloudSave

הכנה:
  1. הרשמה לענן של יוניטי
  2. Edit - Project Settings - Services - Unity Project ID
  3. Window - Package Manager - Unity Registry - Authentication
  4. Window - Package Manager - Unity Registry - Cloud Save
ביצוע:
  1. UnityServices.InitializeAsync
  2. AuthenticationService.Instance.SignInAnonymouslyAsync
  3. AuthenticationService.Instance.SignedIn += async () => { /* load */ };
  4. load: CloudSaveService.Instance.Data.Player.LoadAsync
  5. save: CloudSaveService.Instance.Data.Player.SaveAsync
