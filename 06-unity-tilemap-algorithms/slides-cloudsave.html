שמירת נתוני שחקן - Unity CloudSave

אראל סגל-הלוי

המטרה

אנחנו רוצים לשמור את הנתונים של שחקן מסויים לאורך זמן, למשל כדי לעקוב אחר ההתקדמות שלו.

דרכים אפשריות

  1. בזיכרון - ייעלם לאחר שהשחקן סוגר את המשחק;
  2. בקובץ במחשב המקומי - ייעלם אם השחקן עובר לשחק במחשב אחר ;
  3. במסד נתונים - דורש לנהל שרת פרטי.

הפתרון של יוניטי

CloudSave - שמירת נתוני שחקן על הענן של יוניטי.

  1. Window - Package Manager - Unity Registry - Authentication
  2. הרשמה לענן של יוניטי
  3. Edit - Project Settings - Services - Unity Project ID
  4. Window - Package Manager - Unity Registry - Cloud Save
